接口名称

1) 请求地址

https://tg-message.vercel.app/api

2) 调用方式：HTTP get post

3) 接口描述：

  • TG版消息推送机器人,基于 vercel 函数搭建,免服务器
  • 关注机器人 MessageBot
  • 发送 /token 获取 token
  • Github: https://github.com/anhao/TgMessage

4) 请求参数:

字段名称 字段说明 类型 必填 备注
token string Y token
message string Y 要发送的消息内容

5) 请求返回结果:

{"code":200,"message":"success"}
